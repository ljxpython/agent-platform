## 🎯 修复概述

修复了AI核心框架文档中的`<augment_code_snippet>`标签问题，将所有不能识别的代码片段标签替换为标准的Markdown代码块格式。

## 🔧 问题描述

在AI核心框架文档中，存在大量的`<augment_code_snippet path="..." mode="EXCERPT">`标签，这些标签在标准Markdown渲染器中无法正确显示，影响文档的可读性。

### 问题示例

**修复前：**
```html
<augment_code_snippet path="backend/ai_core/message_queue.py" mode="EXCERPT">
````python
async def get_streaming_sse_messages_from_queue(
    conversation_id: str,
) -> AsyncGenerator[str, None]:
    # 代码内容...
````
</augment_code_snippet>
```

**修复后：**
```markdown
```python
async def get_streaming_sse_messages_from_queue(
    conversation_id: str,
) -> AsyncGenerator[str, None]:
    # 代码内容...
```
```
